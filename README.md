This is an R extension for calculating a researcher's [Kardashian Index](https://en.wikipedia.org/wiki/Kardashian_Index). 

Installation: 
```r
devtools::install_github("gjgetzinger/KaRdashianIndex")
```

Usage: 
```r
k_index(
   twitter_handle = "gjgetzinger", 
   first_name = "gordon", 
   last_name = "getzinger"
)
```
